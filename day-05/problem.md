## Day 5: Longest Palindromic Substring

### Problem Statement:
Given a string `s`, return the **longest palindromic substring** in `s`.

### Examples:
Input: s = "babad"  
Output: "bab"  
(Note: "aba" is also valid)

Input: s = "cbbd"  
Output: "bb"

### Constraints:
- 1 <= s.length <= 1000  
- s consists of only digits and English letters.
